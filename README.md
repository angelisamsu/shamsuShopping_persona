# Shamsu
Basit bir Flutter alışveriş uygulaması. Güzellik ürünlerini listeler, arama yapmaya izin verir ve sepete ürün eklemeyi destekler.

# Flutter Sürümü
Flutter 3.44.4 

# Özellikler
Ürün keşif ekranı

Ürün arama

Ürün detay sayfası

Sepet ve ürün sayısı gösterimi

State management için Provider kullanımı

# Gereksinimler
Flutter SDK kurulu olmalı

Bağlı bir cihaz veya emülatör

# Uygulamayı Çalıştırma
Proje kök dizininde bir terminal açın

Bağımlılıkları yükleyin:
```bash
   flutter pub get

Uygulamayi calistir (spesifik cihazda) :
   ```bash
   flutter devices
   flutter run -d <device_id>
   ```
